# __*args__  
* arguments 약자로 보통 함수 정의 시 입력 사이즈의 제한을 없애기 위해서 사용. 리스트를 전할할 시 unpacking 해서 myfunc(*List) 식으로 전달
***
# __**kwargs__  
* 위와 동일하나 키워드를 여러개 받는 경우활용. myfunc(x=1,y=2,z=3) 등의 형식으로 사용. 하나의 로컬 딕셔녀리를 생성하여 x,y,z 는 키로 1,2,3은 value 로 활용하게 된다.
